Pre-requities

To add the git repository to our local terminal (ssh)
```sh
echo "# <repo_name>" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin git@github.com:<user_name>/<repo_name>.git
git push -u origin main
```

Introduction 
===========================
